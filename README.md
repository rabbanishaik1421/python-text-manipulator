# Python Text Manipulator

A simple command-line based Text Manipulator application developed using Python.  
This project performs multiple text operations using Python string methods and functions.

## Features

The application supports the following operations:

1. Convert text to Uppercase
2. Convert text to Lowercase
3. Convert text to Capital Case
4. Count Characters
5. Count Words
6. Remove Extra Spaces
7. Reverse Text
8. Check if Text is Alphanumeric
9. Check if Text Ends With a Specific Word
10. Center Align Text
11. Count Occurrences of a Word
12. Find Index of a Word
13. Split Text
14. Partition Text
15. Replace Text
16. Exit Program
